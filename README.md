# PRODIGY_ML_01
End-to-end house price prediction system using Python, Scikit-learn, and Kaggle dataset
# 🏠 House Price Prediction

## 📌 Overview

This project predicts house prices using machine learning techniques based on features such as area, number of rooms, and other property attributes.

---

## 📊 Dataset

* Source: Kaggle
* Dataset: *House Prices – Advanced Regression Techniques*

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

---

## 🚀 Features

* Data preprocessing (handling missing values & encoding)
* Feature scaling
* Ridge Regression model implementation
* Model evaluation using RMSE and R² score
* Data visualization (Actual vs Predicted graph)
* Model saving using Joblib

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python src/model.py
```

---

## 📈 Output

* RMSE Score
* R² Score
* Scatter Plot: Actual vs Predicted Prices

---

## 💾 Model

The trained model is saved as:

```
model.pkl
```
